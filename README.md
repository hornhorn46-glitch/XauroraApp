# WittyApp

**WittyApp** — это простое Android‑приложение, разработанное на языке Kotlin с использованием Jetpack Compose. Приложение показывает текущее время и случайно выбирает одну из остроумных фраз. Нажатие на кнопку «Обновить» обновляет дату/время и выводит новую фразу.

## Структура проекта

Проект использует стандартную иерархию Gradle:

* `settings.gradle.kts` — определяет имя проекта, подключает репозитории и включает модуль `app`.
* `build.gradle.kts` (в корне) — содержит общие задачи, например очистку сборки.
* `app/build.gradle.kts` — настройки Android‑модуля: описание `compileSdk`, `minSdk`, включение поддержки Compose и зависимости.
* `app/src/main/java/com/example/wittyapp/MainActivity.kt` — основной код приложения, реализованный на Jetpack Compose.
* `app/src/main/AndroidManifest.xml` — манифест, где задан главный `Activity` и тема.
* `app/src/main/res/values/strings.xml` — ресурсы строк. Также в `res/mipmap-*` находятся иконки приложения.

## Как собрать APK

Для сборки требуется установленный Android SDK, Gradle (или Gradle Wrapper) и JDK. Вы можете собрать APK из командной строки, выполнив следующие шаги:

1. Клонируйте репозиторий:

   ```bash
   git clone <адрес-репозитория>
   cd witty_app
   ```

2. Если вы используете Gradle Wrapper, убедитесь, что у вас есть права на его выполнение (`chmod +x gradlew`), затем соберите debug‑версию APK. В противном случае используйте установленный Gradle (`gradle`):

   ```bash
   ./gradlew assembleDebug
   ```

   Эта команда создаст файл `app-debug.apk` в каталоге `app/build/outputs/apk/`. Согласно документации Android, задача `assembleDebug` собирает и подписывает APK для отладки 【235968387771123†L503-L510】.

3. (Необязательно) Чтобы построить и установить приложение на подключённое устройство или эмулятор, можно использовать задачу `installDebug` 【235968387771123†L503-L517】:

   ```bash
   ./gradlew installDebug
   ```

   При этом APK будет автоматически установлено на запущенный эмулятор или физическое устройство.

Для сборки release‑версии APK потребуется настроить подпись. Дополнительную информацию можно найти в официальной документации Android.

## Импорт в Android Studio

Вы также можете открыть проект в Android Studio (версия 2025.3.1 или новее):

1. Запустите Android Studio и выберите **Open** (Открыть).
2. Укажите путь к каталогу `witty_app`.
3. После синхронизации Gradle вы сможете запустить приложение на эмуляторе или устройстве.
